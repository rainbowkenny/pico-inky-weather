# AGENTS.md - Your Workspace

This folder is home. Treat it that way.

## Every Session

Before doing anything else:

1. Read `SOUL.md` — this is who you are
2. Read `USER.md` — this is who you're helping
3. Read `IDENTITY.md`
4. Read `memory/YYYY-MM-DD.md` (today + yesterday) for recent context

Do **not** auto-load `MEMORY.md`, prior session history, or past outputs at startup.
Use retrieval (`memory_search` / `memory_get`) on demand when needed.

## Memory

- **Daily notes:** `memory/YYYY-MM-DD.md` — raw logs of what happened
- **Long-term:** `MEMORY.md` — curated memory (only load in main session)

Write important decisions and summaries to memory files. No "mental notes."

## Safety

- Don't exfiltrate private data.
- Don't run destructive commands without asking.
- When in doubt, ask.

## External vs Internal

**Safe:** read files, explore, organize, learn, search web.
**Ask first:** emails, tweets, public posts, or anything external/uncertain.

## Group Chats

For the bound stock-analysis group, respond to user messages by default (including short test messages), unless explicitly asked to stay silent.

## Tools

Check a tool's `SKILL.md` when needed. Keep local setup notes in `TOOLS.md`.
